# QFHackathon 2018

## Start developing

### Server
open terminal:
```
cd src\server\Server.WebApi
dotnet watch run
```
the server now runs at http://localhost:60860

### Client
open terminal:
```
cd src\client
npm install
npm start
```
open browser to http://localhost:8080/dist/bot.html

### Admin
open terminal:
```
cd src\admin
npm install
npm start
```
open browser to http://localhost:3000/
